FAQs

How do I get started?

Your Discord server will need the Polycule Graph Bot already on it. If it doesn't contact your Discord admin. If you are the admin, click here to install

How do I add myself to the graph?

Use the /register command in Discord. You can use tab to bring up more options like pronouns.

How do I update my information (pronouns, etc) or existing relationship?

You can run the previous commands again with the updated information.

How do I remove a relationship?

You can use the /remove_partner command in Discord. Use either the partner or partner_name parameter in the same way you used /add_partner

How do I remove myself from the graph?

You can use the /unregister command in Discord.

How can I add/modify relationships which I'm now a part of?

Your Discord admin will have to grant you the Polygrapher. Once you have that, you can use the /add_others_relationship command in Discord.
